## Quick CSS Snippets

### Gradient Button
```css
background: linear-gradient(135deg, #6366f1 0%, #4f46e5 100%);
box-shadow: 0 4px 6px rgba(99, 102, 241, 0.3);
transition: all 0.3s ease;
```

### Card Style
```css

border-radius: 1rem;
padding: 2rem;

```

### Hover Effect
```css
transition: transform 0.3s ease;
```
